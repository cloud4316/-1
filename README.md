# AlgorithmMaster — Учебный сайт ОАИП

## Быстрый старт (Windows, Git Bash)

```bash
bash install.sh    # Установка всего необходимого
bash start.sh      # Запуск сервера
```

## Доступ

| URL | Описание |
|-----|----------|
| http://127.0.0.1:8000/ | Главная страница |
| http://127.0.0.1:8000/login/ | Вход |
| http://127.0.0.1:8000/admin/ | Django Admin |
| http://127.0.0.1:8000/teacher/ | Панель преподавателя |

## Стандартный администратор

- **Логин:** `admin admin` (поле «Фамилия Имя»)
- **Пароль:** `admin`
- **Смените пароль** после первого входа!

## Регистрация студентов

1. Студент заполняет форму на /register/
2. Преподаватель одобряет в /teacher/
3. Студент входит по «Фамилия Имя» + пароль

## Структура курса

- **14 модулей теории** (~72 академических часа)
- **15 практических работ** с встроенным редактором кода
- **5 тестов** с разбором ответов
- **Рейтинг** студентов по группам

## Стек технологий

- Python 3.10+, Django 5.2.6
- SQLite (WAL-режим)
- Waitress (production WSGI)
- WhiteNoise (статика)
- CodeMirror 5 (редактор кода)
