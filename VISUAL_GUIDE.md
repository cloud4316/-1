# AlgorithmMaster — Визуальное описание интерфейса

## 🎨 Дизайн-система

### Цветовая палитра
- **Primary**: #6366f1 (индиго) — кнопки, акценты, ссылки
- **Success**: #10b981 (зелёный) — зачтённые работы, успех
- **Warning**: #f59e0b (оранжевый) — частично зачтено
- **Danger**: #ef4444 (красный) — не зачтено, ошибки
- **Neutral**: серая шкала от #f9fafb до #111827

### Типографика
- **Шрифт**: Inter (Google Fonts via CDN)
- **Моноширинный**: 'JetBrains Mono' для кода
- **Размеры**: base 16px, h1=2rem, h2=1.5rem

### Компоненты
- **Карточки**: белый фон, border-radius 12px, тень при hover
- **Кнопки**: 5 вариантов (primary, success, danger, outline, ghost)
- **Бейджи**: rounded-full, small text, цветной фон
- **Алерты**: border-left 4px, иконка слева

## 📱 Адаптивность

**Desktop (>1024px):**
- Grid 3-4 колонки
- Боковые отступы
- Полная навигация в строку

**Tablet (768-1024px):**
- Grid 2 колонки
- Stat-карточки 2×2

**Mobile (<768px):**
- Grid 1 колонка
- Гамбургер-меню
- Stat-карточки 2×1 (узкие)
- Кнопки на всю ширину

## 🎯 Ключевые фишки дизайна

1. **Градиентные stat-карточки** — визуально привлекательные
2. **Hover-эффекты** — все интерактивные элементы поднимаются при наведении
3. **Иконки-эмодзи** — понятная визуальная навигация
4. **Прогресс-бары** — мотивация студентов
5. **Цветовая индикация статусов** — мгновенное понимание результата
6. **Dark mode** — комфорт для глаз
7. **CodeMirror** — профессиональный редактор кода
8. **Chart.js** — красивые графики для преподавателя

## 🚀 Анимации

- **Появление карточек**: `animate-in` — fadeIn 0.25s
- **Hover карточек**: translateY(-2px) + тень
- **Кнопки**: transform scale при active
- **Прогресс-бары**: transition width 0.4s
- **Уведомления bell**: pop анимация при появлении счётчика
